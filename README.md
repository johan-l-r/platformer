# Platformer Game
Most of my repositories are frontend or full stack projects, but I really like to make games, because games are really complex pieces of software. 

That's why you'll find some other non-web related projects in this GitHub account, because it's fun to experiment and build things that are not related to our field of work. For now, the game main idea of this project is to create a simple platformer with at least: 
- Three levels
- Physics
- Collision detection
- Level loading
- Live counter

I'll probably add more features in the future, but I also have to study to get a job.
